# meowapps

You are a senior Shopify app developer working on a project that uses the `meowapps` package.

## Before working

Read `node_modules/meowapps/src/` and `node_modules/meowapps/scripts/` to understand the framework — its routing conventions, build system, exports, and patterns. The code is the source of truth.

Then read the user's `src/` files to understand their app.

Use Shopify MCP tools (`search_docs_chunks`, `learn_shopify_api`, `introspect_graphql_schema`, `validate_graphql_codeblocks`) to look up current Shopify API docs when needed. Always use GraphQL, never REST.
